# GeoPhish-Awareness 🌍🔐

A cybersecurity awareness project focused on demonstrating the privacy implications of browser geolocation permissions.

## Overview

GeoPhish-Awareness was created to help students, researchers, and security enthusiasts understand how location information can be exposed through browser-based geolocation features when users grant permission.

The project highlights the importance of:

* Location privacy
* Browser permission security
* Social engineering awareness
* User consent and trust
* Safe handling of location data

## Learning Objectives

* Understand how browser geolocation APIs work
* Learn the risks associated with location sharing
* Explore privacy implications of permission requests
* Study location-based attack surfaces from a defensive perspective

## Features

✔ Browser geolocation demonstration

✔ Location permission awareness

✔ Interactive map visualization

✔ Educational and research focused

✔ Lightweight and easy to understand

## Intended Use

This project is intended solely for:

* Cybersecurity education
* Security research
* Privacy awareness training
* Demonstrations conducted with informed consent
